UPLOAD TO GITHUB
1. Extract this ZIP.
2. Open the extracted JPCDesigns-FINAL folder.
3. Select every file inside the folder.
4. In GitHub, open JPCDesign/JPCDesigns.
5. Click Add file > Upload files.
6. Drag all selected files into GitHub.
7. Scroll down and click Commit changes.
8. Cloudflare Pages will redeploy automatically.
